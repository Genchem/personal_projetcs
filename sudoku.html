How to Play

🔢
Fill the Grid

Place numbers 1–9 in every empty cell until the entire 9×9 grid is filled.

↔️
Each Row

Every row must contain all digits 1–9 with no repeats.

↕️
Each Column

Every column must contain all digits 1–9 with no repeats.

⬛
Each 3×3 Box

Each of the nine bold-bordered 3×3 boxes must also contain 1–9 with no repeats.

✏️
Notes Mode

Use Notes to pencil in possible numbers for a cell while you figure it out.

❌
Mistakes

You get 3 mistakes before the game ends and the solution is revealed. Think carefully!

Controls

1–9 Enter a number
Del Erase a cell
N Toggle notes mode
↑↓←→ Move between cells
